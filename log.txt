  セットアップ & デプロイ手順

  1. GitHubにリポジトリを作成

  GitHub (https://github.com/new) で html
  という名前の公開リポジトリを作成してください。（READMEの追加は不要）

  2. ローカルでgit初期化 & プッシュ

  cd /mnt/c/Users/qu_sh/github/html
  git init
  git add index.html
  git commit -m "Initial commit: GitHub Pages sample"
  git branch -M main
  git remote add origin https://github.com/<あなたのユーザー名>/html.git
  git push -u origin main

  3. GitHub Pages を有効化

  1. GitHubのリポジトリページ → Settings → 左メニューの Pages
  2. Source を Deploy from a branch に設定
  3. Branch を main、フォルダを / (root) に設定して Save

  4. サイトにアクセス

  数分後に以下のURLで公開されます:

  https://<あなたのユーザー名>.github.io/html/

  ---
  index.html にはグラデーション背景のカード型デザインのサンプルページを用意しま
  した。プッシュ後にURLにアクセスすると確認できます。
